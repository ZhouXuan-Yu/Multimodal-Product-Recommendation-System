# 多模态商品推荐系统 - GUI界面

## 项目概述

这是一个基于PyQt6开发的现代化电商风格商品推荐系统界面，实现了用户登录、商品浏览、搜索、收藏等核心功能。

## 功能特性

### 🎨 界面设计
- **现代化UI**: 采用简洁的电商风格设计
- **响应式布局**: 支持窗口大小调整
- **美观卡片**: 商品以精美卡片形式展示

### 🔧 核心功能
- **用户登录**: 支持多用户登录和注册
- **商品浏览**: 分类展示和瀑布流布局
- **智能搜索**: 实时商品搜索功能
- **收藏管理**: 商品收藏和取消收藏
- **统计面板**: 数据可视化和图表展示
- **导航系统**: 左侧导航栏快速切换分类

### 📊 数据展示
- **商品卡片**: 显示图片、标题、价格、评分、描述
- **统计图表**: 点击率趋势、偏好分布、热门商品
- **实时更新**: 界面数据动态刷新

## 项目结构

```
├── main.py                 # 主程序入口
├── requirements.txt        # 依赖包列表
├── ui/                     # 界面模块
│   ├── __init__.py
│   ├── styles.py          # 样式定义
│   ├── login_window.py    # 登录窗口
│   ├── main_window.py     # 主窗口
│   ├── product_card.py    # 商品卡片组件
│   └── stats_panel.py     # 统计面板
└── resources/             # 资源文件目录
```

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 默认账户

- 用户名: `admin`, 密码: `admin`
- 用户名: `user1`, 密码: `123456`

## 界面说明

### 登录界面
- 支持用户名密码登录
- 提供注册新用户功能
- 显示演示账户信息

### 主界面布局

#### 左侧导航栏
- 用户信息显示
- 分类导航：首页推荐、电子产品、服装鞋包、家居用品等
- 个人功能：我的收藏、推荐历史、偏好设置

#### 顶部搜索区域
- 商品关键词搜索
- 实时搜索结果过滤
- 刷新按钮

#### 中央商品展示区
- 瀑布流商品卡片布局
- 每行显示3个商品
- 支持滚动浏览

#### 右侧统计面板
- 推荐数据统计卡片
- 点击率趋势图表
- 用户偏好分布饼图
- 热门商品Top10柱状图

### 商品卡片功能
- 商品图片展示
- 基本信息显示（标题、价格、评分）
- 描述预览
- 查看详情按钮
- 收藏/取消收藏功能
- 鼠标悬停效果

## 技术实现

### PyQt6特性
- 信号槽机制处理用户交互
- QThread支持异步操作
- 自定义样式表实现美观界面
- 响应式布局设计

### 数据管理
- 内置示例商品数据
- 模拟用户行为记录
- 分类过滤和搜索功能

### 可视化组件
- Matplotlib集成图表展示
- 实时数据更新机制
- 多种图表类型支持
