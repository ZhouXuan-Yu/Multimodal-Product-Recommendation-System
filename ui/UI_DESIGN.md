# 多模态商品推荐系统 - 现代化界面设计

## 🎨 设计理念

参考淘宝、京东、亚马逊等主流电商平台，打造现代化、用户友好的推荐系统界面。

### 设计原则
- **现代化**: 采用扁平化设计、圆角元素、阴影效果
- **用户友好**: 直观的导航、清晰的信息层次、流畅的交互
- **响应式**: 自适应布局、合适的间距和字体大小
- **品牌一致**: 统一的配色方案和视觉元素

## 🎛️ 交互设计

### 导航交互
- **顶部导航**: 点击分类按钮切换商品展示
- **搜索功能**: 实时搜索商品标题和描述
- **用户菜单**: 显示用户信息和操作选项

### 商品交互
- **卡片悬停**: 显示阴影和微动画效果
- **点击查看**: 弹窗显示商品详细信息
- **收藏功能**: 按钮状态切换和视觉反馈

### 数据展示
- **统计面板**: 独立窗口，展示多维度数据
- **图表可视化**: Matplotlib集成，现代化图表样式
- **实时更新**: 数据动态刷新和状态同步

## 📱 响应式设计

### 窗口尺寸适配
- **大屏 (1600x1000)**: 5列商品布局
- **中屏 (1200x800)**: 4列商品布局
- **小屏 (800x600)**: 3列商品布局

### 字体和间距
- **基础字体**: Microsoft YaHei UI, 13px
- **标题字体**: 16-20px, 加粗
- **间距系统**: 8px基础单位，倍数递增

## 🔧 技术实现

### PyQt6特性
```python
# 现代化样式表
CARD_STYLE = """
QWidget {
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.06);
}
QWidget:hover {
    box-shadow: 0 4px 16px rgba(255, 107, 53, 0.15);
    transform: translateY(-2px);
}
"""

# 信号槽机制
card.clicked.connect(self.on_product_clicked)
nav_btn.clicked.connect(lambda: self.on_category_clicked(category))
```

### 布局管理
- **QVBoxLayout/QHBoxLayout**: 主布局框架
- **QGridLayout**: 商品卡片网格
- **QScrollArea**: 商品列表滚动
- **QSplitter**: 可调整大小的面板

### 样式系统
- **动态样式**: 根据状态变化更新样式
- **主题支持**: 预留主题切换接口
- **一致性**: 统一的颜色和字体变量
